# Landing Page

1. Review [T3 Code](https://github.com/pingdotgg/t3code) and check what it uses for its website and landing page, including its stack and style.
2. Review [Executor](https://github.com/UsefulSoftwareCo/executor) and check how it implements its website and landing page.
3. Take the best of both and create a similar page, in stack and style, for the current project. Ask the user when you are not sure which direction to take.
4. Create a GitHub Actions workflow and publish the page with GitHub Pages.
